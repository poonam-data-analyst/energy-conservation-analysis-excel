# energy-conservation-analysis-excel

# PROJECT OVERVIEW

This project analyzes key factors influencing energy conservation in the cement industry using Interpretive Structural Modeling (ISM). Microsoft Excel was used to perform calculations, construct matrices, and analyze relationships between different drivers of energy efficiency.

# TOOLS USED

1)Microsoft Excel

2)Data Analysis

3)Interpretive Structural Modeling (ISM)

4)Project Objective

The objective of this project is to identify critical drivers of energy conservation and understand their hierarchical relationships to support better decision-making in industrial energy management.

# Dataset / Model Components

1)The Excel model includes the following analytical components:

2)Structural Self-Interaction Matrix (SSIM)

3)Initial Reachability Matrix

4)Final Reachability Matrix

5)Level Partitioning

6)Hierarchical Structure Modeling

# Key Analysis Performed

1)Identification of major factors affecting energy conservation

2)Construction of relationship matrices between influencing variables

3)Determination of hierarchical levels of drivers

4)Visualization of interdependencies among factors

# Skills Demonstrated

1)Data Analysis using Excel

2)Matrix Modeling

3)Industrial Data Interpretation
